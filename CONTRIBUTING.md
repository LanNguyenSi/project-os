# Contributing to project-os

Thanks for your interest. project-os is the umbrella for the Project OS suite (project-forge, agent-tasks, agent-preflight, etc.). This repo holds the README map, architecture diagrams, docs, and the module registry.

## Issues

Most code-level bugs and feature requests belong in the **module's own repo** (links in [README.md](README.md)). Open issues here for:

- README / diagram / docs corrections
- Cross-module questions about how the suite fits together
- Module-registry entries (`registry/`)

## Pull Requests

1. Fork, branch off `master` (e.g. `docs/<scope>`).
2. Keep changes scoped (typo, diagram fix, registry entry, etc.).
3. Open the PR with a clear summary, motivation, and reference to any module repo it relates to.

## Style

Match the surrounding tone. Prefer small, reviewable diffs.
